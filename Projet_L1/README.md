# Théorie – Lentilles minces

## 1. Lentille convergente
- Centre plus épais que les bords  
- Focale **positive**  
- Les rayons parallèles convergent vers le foyer image  

## 2. Lentille divergente
- Centre plus fin que les bords  
- Focale **négative**  
- Les rayons parallèles divergent ; leur prolongement passe par le foyer objet  

## 3. Formule de conjugaison
\[
\frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}
\]

- \( f \) : focale  
- \( d_o \) : distance objet-lentille  
- \( d_i \) : distance image-lentille (positive = image réelle, négative = virtuelle)

## 4. Grandissement
\[
g = \frac{h_i}{h_o} = -\frac{d_i}{d_o}
\]

- Signe – : image renversée si réelle  
- \(|g| > 1\) : image agrandie
